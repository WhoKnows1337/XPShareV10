# Tech Stack

## Frontend
- **Next.js 15.1.5** (App Router, Server Components, React 19)
- **TypeScript 5** (strict mode enabled)
- **Tailwind CSS 3.4** + tailwindcss-animate
- **shadcn/ui** + Radix UI primitives
- **Framer Motion 11** (animations)
- **next-intl 3.26** (i18n with 4 languages: DE, EN, FR, ES)
- **Lucide React** (icons)

## Backend & Database
- **Supabase** (PostgreSQL + Auth + Storage + Realtime + RLS)
  - @supabase/supabase-js 2.49
  - @supabase/ssr 0.7 (Server-Side Rendering support)
  - @supabase/auth-helpers-nextjs 0.10
- **Neo4j Aura** (Graph database for pattern detection)
  - neo4j-driver 5.28
- **OpenAI API** (GPT-4o-mini, Whisper, Embeddings)
  - openai 4.77

## Key Libraries
- **React Hook Form 7.54** + Zod 3.24 (form validation)

- **Mapbox GL 3.15** + react-map-gl 7.1 (interactive maps)
- **Recharts 3.2** (analytics charts)
- **date-fns 4.1** (date formatting)
- **Zustand 5** (state management)
- **SWR 2.2** (data fetching)
- **TipTap 2.10** (rich text editor)
- **RecordRTC 5.6** (audio recording)
- **Sonner 2** (toast notifications)
